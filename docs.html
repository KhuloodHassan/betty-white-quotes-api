API

The Betty White Quotes API provides a collection of memorable quotes from the beloved actress and comedian Betty White. The API consists of keyed objects that will return a quote, tag, and author.

Calling the API

Authentication

No authentication is required to use the Betty White Quotes API.

API response format

                
                    {
                        "author": "Betty White",
                        "tag": "Humor",
                        "quote": "I'm a health nut. My favorite food is hot dogs with French fries. And my exercise: I have a two-story house and a very bad memory, so I'm up and down those stairs."
                    }

Endpoints

To fetch a quote from the API, simply make a GET request to the endpoint. The response will be a JSON object with the following properties:

  • quote: The text of the quote
  • tag: A string indicating the context or topic of the quote
  • author: The name of the person who said the quote (in this case, Betty White)

Fetch a random quote

GET https://bettywhite.cyclic.app/api/random fetch a random quote.

Fetch a quote by id

GET https://bettywhite.cyclic.app/api/{id} fetch a quote by id. id's range from 1 to 117.

Fetch all quotes

GET https://bettywhite.cyclic.app/api/all fetch all quotes.

Attribution

If you use the Betty White Quotes API in your project or application, we ask that you include attribution to Betty White as the source of the quotes.
